Task 3 - Contact Book

Description

This project is a simple Contact Book application developed using Python and Tkinter. It allows users to store and manage contact information, including name, phone number, email, and address. Users can add, view, search, update, and delete contacts through a user-friendly graphical interface.

Features

* Add new contacts
* View all saved contacts
* Search contacts by name or phone number
* Update existing contact details
* Delete contacts
* Automatically save contact information
* Easy-to-use graphical user interface

Technologies Used

* Python
* Tkinter

Files

* contact_book.py – Main application
* contacts.txt – Stores contact details
* README.md – Project documentation

How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the application using:

python contact_book.py

How to Use

1. Enter the contact's Name, Phone Number, Email, and Address.
2. Click Add Contact to save the contact.
3. Click View All Contacts to display all saved contacts.
4. Enter a name or phone number in the search box and click Search Contact.
5. Select a contact from the list, edit the details, and click Update Contact.
6. Select a contact and click Delete Contact to remove it.
