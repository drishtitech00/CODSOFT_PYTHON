Task 2 - Rock Paper Scissors Game
Description

This project is a simple Rock Paper Scissors game developed using Python and Tkinter. The user selects Rock, Paper, or Scissors, and the computer randomly chooses one of the three options. The application then displays the result and updates the scores.

Features
User-friendly graphical interface
Play Rock, Paper, or Scissors
Random computer choice
Displays both user and computer choices
Shows whether the user wins, loses, or the game is a tie
Keeps track of user and computer scores
Reset game option
Technologies Used
Python
Tkinter
Random Module
Files
rock_paper_scissors.py – Main Python program
README.md – Project documentation
How to Run
Install Python.
Open the project folder.
Run the following command:

python rock_paper_scissors.py

How to Play
Click on Rock, Paper, or Scissors.
The computer will automatically make a random choice.
The application displays both choices.
The winner is decided according to the game rules.
Scores are updated after every round.
Click Reset Game to start a new game.
Game Rules
Rock beats Scissors.
Scissors beat Paper.
Paper beats Rock.
If both players choose the same option, the game is a tie.
