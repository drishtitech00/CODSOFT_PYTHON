Task 1 - To-Do List Application

Description
This is a simple To-Do List application developed using Python and Tkinter. It allows users to manage their daily tasks through an easy-to-use graphical user interface (GUI).

Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Automatically save tasks to a text file (`tasks.txt`)
- Load previously saved tasks when the application starts
- User-friendly graphical interface

Technologies Used
- Python
- Tkinter

Files
```
todo.py        - Main application
tasks.txt      - Stores all tasks automatically
README.md      - Project documentation
```

How to Run
1. Make sure Python is installed.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python todo.py
```

How to Use
1. Enter a task in the text box.
2. Click Add Task to add it to the list.
3. Select a task and click Complete Task to mark it as completed.
4. Select a task and click Delete Task to remove it.
5. All tasks are automatically saved and loaded using `tasks.txt`.

Project Output
The application provides a simple GUI where users can:
- Add tasks
- Complete tasks
- Delete tasks
- View all saved tasks
